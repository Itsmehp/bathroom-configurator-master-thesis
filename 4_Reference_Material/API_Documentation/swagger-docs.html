Eckeinstieg

Search for ECKEINSTIEG doors and compatible NICHE doors

Searches for doors where the given width falls within the minEinbau/maxEinbau range. When isShortSidewallEnabled is false, shows all ECKEINSTIEG doors plus NICHE doors with relationships to seitenwand that have isShortSidewall=false. When isShortSidewallEnabled is true, only shows doors that have relationships with seitenwand that have isShortSidewall=true.

Authorizations:
bearerAuth
Request Body schema: application/json
required
width
required
number

Desired width in mm (will be matched against minEinbau/maxEinbau with ±50mm tolerance)

orientation
string

Orientation (LEFT, RIGHT, UNIVERSAL). If LEFT/RIGHT, include UNIVERSAL as compatible.

openingType
string

Door opening type (e.g., GLEITUR, PANDELTUER). Optional.

isShortSidewallEnabled
boolean

Whether to filter for products compatible with short sidewalls. Optional.

Responses

http://localhost:8000/api/eckeinstieg/search

Request samples

  • Payload
Content type
application/json
{
  • "width": 1200,
  • "orientation": "LEFT",
  • "openingType": "GLEITUR",
  • "isShortSidewallEnabled": false
}
